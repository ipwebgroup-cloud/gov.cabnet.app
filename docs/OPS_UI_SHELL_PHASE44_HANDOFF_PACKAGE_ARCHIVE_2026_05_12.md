# Ops UI Shell Phase 44 — Handoff Package Archive

Date: 2026-05-12

## Purpose

Adds an admin-only private archive page for safe handoff ZIP packages generated by the CLI runner.

Route:

```text
/ops/handoff-package-archive.php
```

## Safety

The page does not build packages and does not dump the database.

It only lists, downloads, or deletes whitelisted generated ZIP files from:

```text
/home/cabnet/gov.cabnet.app_app/var/handoff-packages
```

It blocks path traversal by requiring filenames matching:

```text
gov_cabnet_safe_handoff_YYYYMMDD_HHMMSS_with_db.zip
gov_cabnet_safe_handoff_YYYYMMDD_HHMMSS_no_db.zip
```

## Operational warning

Packages marked `WITH DATABASE` contain `DATABASE_EXPORT.sql` and should be treated as private operational material. Do not commit database exports to GitHub unless intentionally sanitized.

## Production behavior

No Bolt calls. No EDXEIX calls. No AADE calls. No queue staging. No live submission changes. The production pre-ride email tool is unchanged.
